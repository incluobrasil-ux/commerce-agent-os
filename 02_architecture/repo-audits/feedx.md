# google-marketing-solutions/feedx

- **Objetivo:** framework para experimentação (A/B) de mudanças em feeds de produto.
- **Por que foi selecionado:** complementa feedgen — sem experimentação, mudanças no feed são apostas cegas.
- **Papel no projeto:** referência metodológica para o eixo `02_architecture/experimentation`.
- **Categoria no monorepo:** `01_upstreams/feedx` (read-only).
- **Modo de uso:** referência.
- **Risco / limitação:** Python + dependências analíticas pesadas; só faz sentido após termos volume de tráfego/conversões para experimentar.
- **Prioridade:** média (depois que feedgen estiver rodando).
- **Status local:** não clonado.
- **Notas a verificar:** desenho estatístico usado; granularidade de evento exigida.
