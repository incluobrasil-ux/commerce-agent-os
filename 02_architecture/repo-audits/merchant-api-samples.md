# google/merchant-api-samples

- **Objetivo:** exemplos oficiais de uso da nova Merchant API (sucessora da Content API for Shopping).
- **Por que foi selecionado:** referência canônica para integrar feeds, produtos e relatórios do Google Merchant Center.
- **Papel no projeto:** referência + integração; código de referência informa o adapter em `05_integrations/google-merchant`.
- **Categoria no monorepo:** `01_upstreams/merchant-api-samples` (read-only) + adapter autoral em `05_integrations/google-merchant`.
- **Modo de uso:** referência + integração via SDK oficial.
- **Risco / limitação:** OAuth + setup de Merchant Center; rate limits; migração Content API → Merchant API ainda em curso no ecossistema.
- **Prioridade:** alta.
- **Status local:** não clonado.
- **Notas a verificar:** quais subAPIs já estão GA; linguagens dos samples; preferir a TS/JS quando existir.
