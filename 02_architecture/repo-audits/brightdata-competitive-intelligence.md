# brightdata/competitive-intelligence

- **Objetivo:** kit/exemplos para coleta de inteligência competitiva usando a plataforma Bright Data (datasets, scraping browser, etc.).
- **Por que foi selecionado:** módulo de marketing-research precisa de dados externos (concorrentes, preços, reviews).
- **Papel no projeto:** integração com Bright Data via `05_integrations/brightdata`.
- **Categoria no monorepo:** `01_upstreams/brightdata-competitive-intelligence` (read-only) + adapter autoral.
- **Modo de uso:** integração.
- **Risco / limitação:** serviço **pago**; questões legais sobre scraping (ToS dos sites-alvo, LGPD/GDPR); custo escala com volume.
- **Prioridade:** média (após termos casos de uso concretos).
- **Status local:** não clonado.
- **Notas a verificar:** quais produtos da Bright Data o repo cobre; existe alternativa nativa (APIs públicas) para nossos casos antes de comprar.
