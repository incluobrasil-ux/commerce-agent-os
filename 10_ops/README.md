# 10_ops

Scripts, CI/CD e observabilidade operacional. Tudo que não é código de aplicação nem doc.

## Subdirs

- [`scripts/`](./scripts/) — scripts shell/Node executáveis localmente ou no CI (bootstrap, validate, check-env, etc.).
- [`security/`](./security/) — política operacional de segurança (checklist).
- `ci/` (futuro) — definições de pipelines (GitHub Actions, etc.).
- `higgsfield-cli/` (futuro) — wrapper sobre a CLI da Higgsfield.
- `runbooks/` (futuro) — procedures operacionais para incidentes.

## Status

`scripts/` e `security/` materializados. Demais virão conforme implementação avança.
