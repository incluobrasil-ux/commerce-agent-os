# higgsfield/types/

Tipos do domínio Higgsfield. **Provisórios** — confirmar ao clonar `higgsfield-ai/skills` e `higgsfield-ai/cli`.

## Tipos previstos

- `SkillId` — branded id da skill (`<namespace>/<name>@<version>`).
- `SkillManifest` — metadados (id, description, inputs schema, output schema, tags).
- `SkillExecutionInput` — payload para executar.
- `SkillExecutionResult` — saída + provenance.
