# Repo audits

Um arquivo por repositório estudado. Cada audit tem o mesmo schema curto.

Classificação mestra: [../../00_meta/REPO_SELECTION.md](../../00_meta/REPO_SELECTION.md).

## Schema

```
# <repo>
- Objetivo
- Por que foi selecionado
- Papel no projeto
- Categoria no monorepo
- Modo de uso (upstream | referência | base operacional | integração)
- Risco / limitação
- Prioridade (alta | média | baixa)
- Status local (não clonado | clonado | adaptado)
- Notas a verificar
```
