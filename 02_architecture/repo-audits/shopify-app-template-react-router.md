# Shopify/shopify-app-template-react-router

- **Objetivo:** template oficial de app embedded Shopify com React Router + Polaris + App Bridge.
- **Por que foi selecionado:** caminho mais curto e suportado oficialmente para criar um app embedded no admin Shopify.
- **Papel no projeto:** base operacional para `04_apps/shopify-app`.
- **Categoria no monorepo:** `01_upstreams/shopify-app-template-react-router` (read-only) + cópia inicial em `04_apps/shopify-app`.
- **Modo de uso:** upstream read-only + base operacional (clonar uma vez, evoluir como código autoral).
- **Risco / limitação:** exige conta Shopify Partners, Shopify CLI, túnel (ngrok/cloudflare); chaves OAuth via env.
- **Prioridade:** alta.
- **Status local:** não clonado.
- **Notas a verificar:** versão atual do template; se ainda usa Prisma/SQLite por padrão; convenção de variáveis de ambiente.
